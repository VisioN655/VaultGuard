# VaultGuard – Passwortmanager App

Projekt von:
- Affan Atik (77202085747)

## Beschreibung

VaultGuard ist eine Android-App zur sicheren Verwaltung von Passwörtern.  
Nutzer können Passwörter erstellen, speichern, bearbeiten und löschen.  
Zusätzlich enthält die App einen Passwort-Generator.

## Funktionen

- Registrierung und Login mit Firebase
- E-Mail-Verifizierung
- Passwort hinzufügen, bearbeiten und löschen
- Verschlüsselte Speicherung der Passwörter
- Passwort-Generator (anpassbare Länge & Zeichen)
- Kopieren von Passwörtern in die Zwischenablage

## Sicherheit

Die Passwörter werden vor dem Speichern verschlüsselt (AES)  
und beim Laden wieder entschlüsselt.

## Technologien

- Java
- Android Studio
- Firebase Authentication
- Firebase Firestore
